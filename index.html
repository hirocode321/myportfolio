About Me

フォトグラファーとしての「視点」と、エンジニアとしての「技術」。
二つの領域を繋ぐクリエイターを目指しています。

はじめまして。関岡 大晃（せきおか ひろあき）と申します。
私は2018年3月に関西学院大学経済学部を卒業しました。その後、自動車部品メーカーにて生産管理業務に従事いたしました。製造現場に深く入り込み、現場改善や工程設計を通じて、論理的思考力、数値分析力、そしてプロジェクト推進力を養いました。工場の稼働率を高めるために、関係部署との綿密な調整やシステムの改修にも携わった経験がございます。

2021年1月からは、フリーランスフォトグラファーとして独立いたしました。
天体・風景写真を専門とし、広告撮影や出版活動、講演、セミナー登壇など、幅広く活動してまいりました。特に星景写真の分野では、Sony公式セミナー講師を務めさせていただいたり、CP+などの大規模イベントで登壇したり、星空関連書籍への執筆・写真提供など、数多くの実績がございます。また、SNSでの発信にも力を入れており、X（旧Twitter）では6.5万人、Instagramでは2.3万人を超えるフォロワー様に応援していただいております。地方自治体の観光マーケティングにも携わり、写真の力で地域の魅力を発信してまいりました。

観光マーケティングの経験を経て、これからはITの知識が不可欠であると感じ、創造社デザイン専門学校にてITを広く学び始めました。
デジタルマーケティングに加え、Python (Flask/Django) を用いたWebアプリケーション開発を学習しております。その中でプログラミングの面白さに触れ、エンジニアへの転身を決意いたしました。現在は、これまでの写真家としての経験を活かし、天体撮影を効率化するツールの開発に注力しております。具体的には、SkyfieldやNumPyなどのライブラリを活用した天体位置計算や、ユーザーインターフェース (UI) を意識した設計に挑戦しています。

写真、デザイン、そして論理的思考。これら三つを融合させ、「美しさ」と「使いやすさ」を両立したプロダクトを生み出すことを目指しております。

Projects

LunaTide

Python (Flask) Skyfield NumPy Bootstrap 5

月齢・月の出入り時刻・潮位情報を可視化する、写真家向けのWebアプリケーションです。
観測地と日付を入力するだけで、Skyfieldを用いて月の位置を精密に計算し、潮汐データと合わせて表示します。撮影計画の立案を強力にサポートします。

アプリを見る GitHub

EXPOCALC

Python (Flask) JavaScript Bootstrap 5

星空撮影における「適正露出」を導き出す計算シミュレーターです。
カメラのセンサーサイズやレンズの焦点距離を入力することで、星が点像として写る限界のシャッター速度（500ルール/NPFルール）を自動算出します。

アプリを見る GitHub
